## 📝 About This Repository

This repository contains the practical code developed during the course. Its main goal is to consolidate conditional logic concepts in C#, with emphasis on:

- Decision structures with `if`, `else if`, and `else`
- Control flow with `switch/case`
- Logical and relational operators
- Condition-based problem solving
- Readability and organization of branching logic

## 💻 How to Run the Code

Since this project was built using the **.NET CLI**, you can easily run it from your terminal.

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. Clone this repository and navigate to the desired challenge folder.
3. Run the following command in the terminal:

```bash
dotnet run
```
